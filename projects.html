Air Hockey
Play Game Here - Find on GitHub

This is going to be the Fall 2015 SURGE project. It is a 1-4 player air hockey game that works great with both keyboard & mouse and touch. The setup of the SURGE project is a hockey table sized screen plugged into a raspberry pi running the game on either a browser or as an app for android so you are able to play with friends in a game of virtual life sized air hockey.

© John Redden, Cory Lewis, & Paul Gonzalez-Becerra 2015.

Apis-Bundle (Game Engine)
Find on GitHub

This one has had multiple names, but will stick with the name of Apis. I have been working on this one for well over a year now. Before it was one big libraries designed to aid the programmer on developing a video game, but has branched into also being used as a multi-purpose library for not just video games. Hence the suffix "-bundle". Uses OpenGL, OpenAL, and SDL; although in the form of OpenTK. The bundle is very modularly designed so someone can take the libraries and render DirectX if they wanted and it would work just about the same. It also has the capabilities of grabbing other libraries, running or not, and injecting them into the software.

© Paul Gonzalez-Becerra 2015.
