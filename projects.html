Open Graphing Calculator
Website: www.opengraphingcalculator.com

  Open Graphing Calculator began as an exercise that was used to learn a little about creating web applications. And we are still working on it. The goal is to make use of existing libraries to create something useful in a short amount of time. Feel free to fork the source on GitHub, submit issues, and contribute.

GDToolkit

  GDToolkit or (Game / General) Development Toolkit is an open-source low-level game engine api. It was created for building games on a programming level, while at the same time honing on my own programming skills. It uses OpenGL to render and OpenTK for input. It is still a work in progress, but it will eventually use OpenAL for sound, use of DirectX.
