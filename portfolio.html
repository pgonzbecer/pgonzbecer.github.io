Open Graphing Calculator
Website - GitHub

An open source graphing calculator created by John Redden and I. It is capable of doing calculus, like derivatives, tangent lines, integrals, and riemann's sums. Dynamic function composition and the ability to render graphs such as hyperbolas, parabolas, ellipses, and many more on dynamically moving points. It also renders your input beautifully using MathQuill.

Copyright John Redden & Paul Gonzalez-Becerra 2015.
iThrowRock
GitHub

A simple project to see if my colleagues and I could create an Android app and put it on the Google Play store. The throw the rock at passing-by bottles, avoiding the dangerous flaming molotovs. With each passing level, the bottles get faster and faster.

Copyright John Redden, Cory Lewis, & Paul Gonzalez-Becerra 2015 - 2016.
Air Hockey
GitHub

A virtual air hockey game, which is postponed as a school project for later, that simulates a game for 1 to 4 players. Targetted towards bigger screen mobiles, the later project is going to integrate a hockey table sized touch monitor to be able to play the game with.
